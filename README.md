Algorithm

## Description
This program reads a sentence character by character until it reaches a full stop (".").then calculates:

- The total number of characters in the sentence (length)
- The total number of words in the sentence
- The total number of vowels in the sentence

## Input 
 i love mangoes.

## Output
The program displays:
- Length of the sentence
- Number of words
- Number of vowels

## How it works
- Each character is read one at a time
- Length increases for every character
- A word is counted when a space is found
- A vowel is counted when the character is a, e, i, o, or u

##  Output
Length = 15  
Words = 3  
Vowels = 6
